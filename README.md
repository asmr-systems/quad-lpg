# Quad Low Pass Gate (Eurorack)
## IO
**inputs**: 4 carrier in cv (signal in) (3.5mm mono jacks), 4 modulator in cv (env) (banana jacks)
**outputs**: 4 modulated signal out (3.5mm mono jacks)
